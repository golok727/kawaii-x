# X Markdown Renderer

Chrome extension for rendering Markdown in X.com tweets with syntax highlighting.

## Building

1. **Install dependencies**:
   ```bash
   bun install
   ```

2. **Build the extension**:
   ```bash
   bun run build
   ```

 **For development with auto-rebuild**:
   ```powershell
   bun run dev
   ```
